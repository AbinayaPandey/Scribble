## Packages
react-cropper | For image cropping functionality
cropperjs | Core library for cropping
pdf-lib | For PDF manipulation (extracting pages)
pdfjs-dist | For rendering PDF thumbnails
react-dropzone | For drag and drop file uploads
framer-motion | For smooth UI transitions
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes

## Notes
The application is client-side heavy.
Images and PDFs are processed in the browser.
No backend storage is required for the editing features themselves, but the structure is prepared for it.
